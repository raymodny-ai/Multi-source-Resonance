## 1. 项目背景与目标

### 1.1 项目背景

多源共振监控系统（Multi-source Resonance）是一套面向美股与加密市场量化交易的全自动监控后端，已实现以下核心能力：做市商 GEX Gamma 敞口计算、VIX 期限结构分析、加密衍生品去杠杆监控、机构暗盘三驾马车验证（DIX / Short Volume / Stockgrid 净头寸）、多维共振信号评分（Level 1-3）与触发状态机、Hawkes 过程自激抛售测算，以及多渠道告警推送（Email / Telegram / Discord）等完整后端能力。

当前系统以无头终端模式运行，所有监控数据、信号触发、告警通知仅通过日志文件和消息推送输出。对于量化交易员和风控人员而言，缺乏一个直观、可交互、支持实时刷新的前端 Web UI。

### 1.2 产品目标

| 目标 | 描述 | 衡量标准 |
|------|------|----------|
| 可视化总览 | 用户打开页面 3 秒内获取四大维度当前状态 | 仪表盘加载时间 < 3s |
| 深度探索 | 可下钻查看任一维度历史趋势与明细 | 图表交互响应 < 500ms |
| 信号可回溯 | 完整时间轴展示所有历史共振触发 | 支持 90 天历史查询 |
| 系统可观测 | 一目了然数据源健康度、降级模式、调度日志 | 状态刷新延迟 < 15s |
| 配置即页面 | 阈值/频率/通知在页面上修改即生效 | 配置变更 < 5s 生效 |

### 1.3 前端定位

前端 UI 是**纯展示与轻量配置层**，不承载核心计算逻辑。所有量化计算（GEX、DIX、Hawkes 等）由现有 Python 后端完成，前端通过 RESTful API 读取聚合数据并渲染。


## 2. 用户画像与使用场景

### 2.1 用户画像

**用户 A：独立量化交易员（Primary）**
- 背景：自主管理资金，使用多因子模型指导交易决策
- 技能：熟悉 GEX/DIX 概念，能读懂期权链数据
- 行为：开盘期间每 15-30 分钟扫一眼仪表盘，收盘后深度复盘
- 痛点：需要快速确认是否触发“右侧做多”条件，不想盯 4 个数据源

**用户 B：机构风控分析师（Secondary）**
- 背景：负责监控组合尾部风险，需全天候关注市场微观结构异常
- 技能：精通波动率曲面、期限结构，关注系统性风险信号
- 行为：收到 LEVEL 3 告警后，打开 Web UI 核实各维度详情

**用户 C：系统运维人员（Tertiary）**
- 背景：负责维护监控系统正常运行
- 技能：DevOps，熟悉 Python/Linux
- 行为：每天检查一次系统状态页，排查爬虫失败原因

### 2.2 核心使用场景

| 场景 ID | 场景名称 | 用户 | 频率 | 关键页面 |
|---------|----------|------|------|----------|
| S1 | 盘中快速扫视 | A | 每 15-30 分钟 | 实时仪表盘 |
| S2 | 暗盘深度分析 | A/B | 每日盘后 | 暗盘详情页 |
| S3 | 信号核实与决策 | A/B | 收到告警后 | 共振信号面板 |
| S4 | 历史复盘 | A | 周末 | 信号时间轴 |
| S5 | 数据源故障排查 | C | 每日/故障时 | 系统状态监控页 |
| S6 | 阈值调优 | A/B | 月度 | 参数配置后台 |
| S7 | 手机快速确认 | A/B | 收到 Telegram 后 | 移动端仪表盘 |

## 4. 页面一：多维度实时仪表盘（默认首页）

**优先级**：P0（MVP 必须上线）

### 4.1 功能描述

系统默认首页，聚合展示四大核心维度的实时状态，帮助用户在 3 秒内完成市场微观结构的快速扫描。包含以下子模块：

- **GEX Gamma 敞口卡片**：展示当前 GEX 敞口值（单位 $M）与方向变化趋势（FLIP ON/NEGATIVE）
- **VIX 期限结构卡片**：展示 VIX 期货期限结构状态（Contango/Backwardation）及斜率方向
- **Crypto 杠杆清洗卡片**：展示 BTC 资金费率、未平仓合约量（OI）及去杠杆完成状态（CLEANUP PENDING/COMPLETE）
- **暗盘综合得分卡片**：展示 DIX 暗盘强度百分比 + 卖空比 + Stockgrid 净头寸三选二触发状态
- **共振等级实时横幅**：展示共振总分（0~5.0）及 LEVEL 等级（LEVEL 3/2/1/NO_SIGNAL），等级配色随阈值动态变化
- **Hawkes 衰竭概率进度条**：小字标注分支比（0~∞），含状态判断（SUBCRITICAL / CRITICAL / SUPERCRITICAL）
- **底部最近告警列表**：展示最近 5 条告警，支持点击跳转详情

## 5. 页面二：暗盘数据详情页

**优先级**：P0（MVP 必须上线）

### 5.1 功能描述

深挖机构暗盘三驾马车（DIX / Short Volume / Stockgrid 净头寸）的历史趋势与背离标注。帮助用户在收盘后深度复盘机构资金流向，识别主力吸筹/派发行为。

包含以下核心组件：
- **DIX 走势图**：含底背离标注、45% 阈值虚线
- **卖空比组合图表**：展示 Short Volume 占总量比例的历史趋势
- **Stockgrid 净头寸趋势图**：双线展示 20 日斜率与 60 日斜率，标注 Golden Cross 和底背离点
- **三驾马车综合评分侧边栏**：实时展示当前 DIX、Short Volume、Stockgrid 三条指标的触发状态
- **历史评分热力图**：（P2 迭代）以日历热力图形式展示历史得分密度

### 9.5 验收标准

|-------------|--------|------|

| 403 | `FORBIDDEN` | 无权限访问 |
